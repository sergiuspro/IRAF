Работа с пакетами IRAF

Все команды IRAF содержатся в так называемых пакетах, объединяющих, как правило, тематически связанные между собой команды. Некоторые пакеты содержат в себе другие пакеты. При каждом запуске IRAF в окне командной строки IRAF выводится список всех установленных базовых пакетов. Если открыть (загрузить) один из них, тут же будет выведен аналогичный список его содержимого - команд и/или пакетов. В таких списках пакет обозначен точкой после своего названия, в то время как команда в пакете обозначается названием без точки.

Чтобы какой-то пакет автоматически загружался при каждом запуске IRAF, его название нужно прописать в файле login.cl - по одному названию пакета на строчку. В принципе прописать названия можно где угодно в login.cl - начале, середине или конце, но для удобства и упорядоченности login.cl лучше прописывать их сгруппированно, например, после строки (где по умолчанию уже прописаны пакеты: images, plot, dataio, lists, а чуть ниже также пакеты: tv, utilities, noao, vo):

# List any packages you want loaded at login time, ONE PER LINE.

Чтобы после запуска IRAF загрузить какой-либо пакет вручную, достаточно один раз в командной строке ввести его название (при этом важно помнить, что не каждый пакет можно загрузить, находясь при этом в любом другом пакете, нередко требуется сначала по древовидной структуре пакетов перейти в пакет, содержащий пакет, который нужно загрузить), например:

noao

Просмотреть команды/пакеты текущего пакета:

?

Просмотреть команды/пакеты указанного и уже загруженного пакета:

?dataio

Закрыть загруженный пакет и перейти из него на один уровень выше:

bye

Просмотреть список всех загруженных на данный момент пакетов:

package

Вывести на экран список всех установленных пакетов с перечислением всех команд/пакетов в каждом из них:

??

Узнать, в каком пакете содержится та или иная команда IRAF (используемая для этого команда references находится в пакете system), при этом будет выведен построчный список всех вариантов, в контексте которых встречается указанная команда, а название пакета указано в квадратных скобках в конце строки:

references команда

Вывести помощь для указанных пакетов (если не указать имя пакета, будет выведена помощь для текущего пакета):

help имена-пакетов-через-запятую
