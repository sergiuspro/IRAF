Работа с командами IRAF

Все команды IRAF являются уникальными выражениями в среде IRAF. Пишутся они исключительно строчными буквами. В командной строке IRAF допустимо сокращённое написание команд, если сокращение не является многосмысленным, например, вместо команды imcombine можно написать imcomb, но нельзя написать imco, потому что imco может быть воспринято как и сокращённое написание пакета imcoords.

Документация команд IRAF

Для каждой команды IRAF можно вызвать документацию:

help имя-команды

Чтобы выйти из просмотра документации - нажать на клавишу q; чтобы пролистать текст вниз построчно - нажать Enter или j; чтобы пролистать вниз на пол-экрана - нажать d; чтобы пролистать вниз на полный экран - нажать пробел или f.

Поиск пакетов содержащих искомые команды IRAF

Найти пакет, в котором находится искомая команда IRAF (название пакета будет указано в квадратных скобках в конце строки):

references имя-команды

Если пакеты искомых команд IRAF уже загружены, найти эти пакеты можно командой:

whereis имена-команд-через-пробел

Параметры команд IRAF

У большинства команд IRAF имеются параметры, значения которых влияют на выполнение команды. Поэтому прежде чем выполнить ту или иную команду, всегда следует убедиться в том, что она будет выполнена с нужными значениями параметров, и отредактировать их в случае необходимости.

Просмотреть текущие значения параметров команды:

lparam имя-команды

Отредактировать значения параметров команды:

eparam имя-команды

Редактирование будет происходить в том текстовом редакторе, который указан в файле login.cl как значение переменной editor (в настоящее время в IRAF поддерживаются только 4 редактора: vi, emacs, edt и memacs). По умолчанию это редактор vi, поэтому нужно учесть особенности работы с ним: 1) клавишами "стрелка вверх" и "стрелка вниз" можно перемещаться между параметрами, 2) для изменения значения параметра нужно просто написать его новое значение в строке соответствующего параметра и нажать Enter, после чего новое значение появится вместо старого, 3) чтобы сохранить изменения и вернуться в командную строку IRAF, нужно сначала на клавиатуре набрать символ : (двоеточие), при этом курсор перепрыгнет из строк параметров в нижний левый угол окошка, затем нажать букву q и Enter, 4) чтобы вернуться в командную строку IRAF, но при этом не сохранять изменения, нужно на клавиатуре набрать :q! и Enter, 5) для некоторых параметров можно узнать, какие значения допустимы для них, если в строке данного параметра на клавиатуре набрать ? (знак вопроса) и нажать Enter, тогда внизу окошка появится соответствующая информация.

Если нежеланным образом были изменены значения параметров каких-то команд, в командной строке IRAF можно вернуть значения всех параметров этих команд по умолчанию:

unlearn имена-команд-через-пробел

Если требуется вернуть значения параметров по умолчанию всех команд целого пакета:

unlearn имя-пакета

Если требуется вернуть значения параметров по умолчанию абсолютно всех команд IRAF, достаточно удалить все файлы с расширением .par в директории uparm, которая находится в директории, в которой при установке IRAF была выполнена команда mkiraf (в данном случае полный путь к директории: /home/имя-пользователя/IRAF/uparm). Осуществить эту операцию можно проще всего прямо в командной строке IRAF, последовательно выполнив следующие команды (здесь, home является переменной среды IRAF, её значение можно найти в файле login.cl, и она как раз обозначает директорию, в которой была выполнена команда mkiraf):

cd home$uparm
delete *.par

Ускорение выполнения команд IRAF

Если какие-то команды используются очень часто (например, в циклах или скриптах), для ускорения выполнения этих команд можно внести параметры этих команд в кэш памяти (в ином случае, при каждом выполнении команды её параметры считываются из файла на диске):

cache имена-команд-через-пробел

Примечание: см. команду update.

Создание пользовательких команд IRAF

Создать новую пользовательскую команду IRAF:

task имя-команды
