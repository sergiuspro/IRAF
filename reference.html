Справочник команд IRAF

Команды IRAF в данном справочнике приведены в алфавитном порядке. Указаны: пакет содержащий команду, функция команды, её формат, а также некоторые её параметры с их возможными значениями перечисленными через символ | (после параметра в той же строке начиная с символа # дано описание параметра; значение параметра используемое по умолчанию написано заглавными буквами).

airmass

Пакет: noao.astutil.

Функция: вычислить коэффициент воздушной массы на высоте над горизонтом заданной в градусах или радианах

airmass высота-над-горизонтом

Параметры:

elevation= #высота над горизонтом (=90.0 по умолчанию)

scale= #масштабный фактор земной атмосферы (=750.0 по умолчанию)

radians=yes|NO #задать высоту в радианах (да|НЕТ)

aladin

Пакет: vo.votools.

Функция: выполнить команду проверки статуса работы, запуска или завершения работы программы Aladin

aladin команда

Параметры:

cmd=status|START|stop #команда для выполнения (статус|ЗАПУСТИТЬ|завершить) (без указания команды будет произведён запуск программы Aladin)

bases

Пакет: utilities.

Функция: преобразовать указанное число в другую систему счисления (десятичную, шестнадцатеричную, восьмеричную, двоичную)

bases исходное-число

Параметры:

i= #исходное число (суффикс 'x' у исходного числа обозначает, что оно в шестнадцатеричной системе счисления, суффикс 'b' - в восьмеричной, без суффикса - в десятичной)

nbyte= #количество байтов точности (=0 по умолчанию; допустимые значения: 0, 1, 2 и 4)

verbose=YES|no #отображать названия колонок (ДА|нет)

chkupdate

Пакет: system.

Функция: проверить наличие новой версии IRAF

chkupdate

Параметры:

interval= #кол-во дней интервала между повторными проверками (=0 по умолчанию; значение "-1" отключит автоматическую проверку при запуске IRAF, значение "0" включит автоматическую проверку при каждом запуске IRAF)

clear

Пакет: language.

Функция: очистить окно командной строки IRAF

clear

Параметры: (отсутствуют)

darkcombine не завершены параметры!

Пакет: noao.imred.ccdred.

Функция: сложить и обработать темновые кадры

darkcombine входные-данные

Параметры:

input= #входные данные (файлы, список файлов)

combine=AVERAGE|median #тип сложения после отсеивания (СРЕДНЕЕ|медиана) (при использовании медианы на чётном кол-ве входных данных используется усреднение двух центральных значений)

reject=none|MINMAX|ccdclip|crreject|sigclip|avsigclip|pclip #тип отсеивания (|)

process=YES|no #обработать входные данные перед сложением (ДА|нет)

delete=yes|NO #удалить входные данные после сложения (да|НЕТ)

delete

Пакет: system.

Функция: удалить указанный по имени файл или (при использовании маски в имени) файлы

delete имя-файла

Параметры:

verify=yes|NO #запрос подтверждения у пользователя на удаление (да|НЕТ)

default_action=YES|no #действие по умолчанию (ДА|нет)

allversions=YES|no #удаление всех версий удаляемого файла (ДА|нет)

describe

Пакет: stsdas.

Функция: вывести только раздел DESCRIPTION (описание) из полного текста документации к указанным командам IRAF

Параметры:

dparam наборы-параметров-через-пробел

Параметры:

galactic

Пакет: noao.astutil.

Функция: перевести галактические координаты в экваториальные или наоборот

galactic входные-данные

Параметры:

input= #входные данные (=STDIN по умолчанию; файл, список файлов, консоль STDIN)

in_coords=EQUATORIAL|galactic #тип координат входных данных (ЭКВАТОРИАЛЬНЫЕ|галактические)

print_coords=YES|no #вывести координаты входных данных вместе с координатами выходных (ДА|нет)

Примечание: если не указывать количество выводимых команд, то будет использовано значение по умолчанию; если указать количество выводимых команд, то оно также станет новым значением по умолчанию; если указать количество выводимых команд со знаком минус, то текущее значение по умолчанию не будет изменено; если указать "-999", то будут выведены все ранее выполненные команды

imheader

Пакет: images.imutil.

Функция: вывести заголовок указанного файла FITS

imheader имена-файлов

Параметры:

images= #имена файлов или имя списка файлов

longheader=yes|NO #вывести расширенную версию заголовка (да|НЕТ)

userfields=YES|no #если longheader=да, то вывести также пользовательские ключевые слова заголовка (ДА|нет)

imstack

Пакет: images.imutil.

Функция: сложить указанные снимки (с увеличением размерности)

history входные-данные результирующий-снимок

Параметры:

images= #входные данные (снимки для сложения)

output= #результирующий снимок сложения

title= #название результирующего снимка сложения (=* по умолчанию, т.е. соответствует названию первого из слагаемыех снимков)

pixtype= #тип данных у пикселей результирующего снимка сложения (=* по умолчанию, т.е. соответствует типу первого из слагаемыех снимков)

Примечание: слагаемые снимки должны быть одинаковой размерности (ключевое слово NAXIS в заголовке файла FITS) и одинакового линейного размера; результирующий снимок получает WCS первого из слагаемыех снимков; размерность (NAXIS) результирующего снимка на 1 больше размерности слагаемыех снимков
